git の使い方に関するメモ

リポジトリ作成手順

[power-station] 
    $ cd /Volumes/Projects/THN
    $ mkdir {project-name}.git
    $ cd {project-name}
    $ git init --bare --shared
[端末] 
    $ git clone ssh://toru@power-station.local:/Volumes/Projects/THN/{project-name}.git {project-name}

リポジトリ作成手順(既に作業ディレクトリがある場合)

[power-station] 
    $ cd /Volumes/Projects/THN
    $ mkdir {project-name}.git
    $ cd {project-name}
    $ git init --bare --shared
[端末] 
    $ git remote add origin ssh://toru@power-station.local:/Volumes/Projects/THN/{project-name}.git
    $ git push -u origin master

ブランチ作成

[端末] 
    $ git checkout -b {branch_name}
    $ git push --set-upstream origin {branch_name}

ユーザ指定でのclone（Github等）

    $ git clone https://KONISHI-Toru@github.com/kazu-kitagawa/GreenStudy.git

リモートブランチを切り換える

    $ git checkout -b konishi origin/konishi

別のリモートブランチにpushする。

    $ git push origin master:konishi/20200107

commit し忘れたファイルの追加（push前のみ）

    $ git add {file_name}
    $ git commit --amend --no-edit

add の取り消し

    $ git reset HEAD {file_name}

SSH公開鍵認証でのGitHubアクセス

    $ git clone git@github.com:KONISHI-Toru/sample

CRLFファイルを扱う場合

    $ git config --local core.autocrlf false
    $ git config --global core.whitespace cr-at-eol
